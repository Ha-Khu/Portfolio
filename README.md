# Personal Portfolio

My personal developer portfolio — a single-page site introducing who I am, the projects I've built, and how to reach me.

**Live demo:** https://ha-khu.github.io/Portfolio

## About

I'm Dávid Plevka, an Applied Informatics student and aspiring junior fullstack developer.
This site is hand-built with plain HTML, CSS and JavaScript (no framework) and shows a
selection of my fullstack projects.

## Tech Stack

- **HTML5** — semantic markup
- **CSS3** — custom properties (design tokens), Flexbox & Grid, responsive layout
- **JavaScript (vanilla)** — scroll-aware navbar, scroll-triggered fade-in, mobile menu
- **Font Awesome** — icons

## Features

- Responsive layout (desktop, tablet, mobile)
- Dark navy / mint theme driven by CSS variables
- Navbar that hides on scroll down and reappears on scroll up
- Scroll-triggered fade-in animations
- Project cards linking to their GitHub repositories

## Run locally

```bash
git clone https://github.com/Ha-Khu/Portfolio.git
cd Portfolio
```

Then open `index.html` in your browser, or serve it with a local server
(e.g. the VS Code **Live Server** extension) for the cleanest experience.
